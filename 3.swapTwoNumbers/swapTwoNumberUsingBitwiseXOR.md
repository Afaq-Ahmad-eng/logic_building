# Swap two numbers using bitwise XOR

We use the bitwise XOR to solve this swap two numbers problem 

For example : a = 23, b = 13,
when we do a = a ^ b, 23 ^ 13 then a will be become 26
b = a ^ b, 26 ^ 13 then b will be become 23
a = a ^ b, 26 ^ 23 then a will be become 13

Time Complexity is O(1)
Space Complexity is O(1)
